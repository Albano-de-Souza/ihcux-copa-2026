# ihcux-copa-2026
Projeto CopaNaMão - IHC e UX
Nome: Albano de Souza

Problema Priorizado
O foco deste protótipo foi melhorar a experiência do torcedor dentro da arena, atacando três dores principais: o tempo perdido em filas de comida, a dificuldade de localização nos setores e a perda de lances importantes do jogo.

Justificativa de Design
Para que o aplicativo seja funcional em um ambiente de estádio (com barulho, sol forte e agitação), as telas foram pensadas da seguinte forma:

Interface Simples: Usei o modelo de baixa fidelidade com componentes grandes para garantir que o torcedor consiga clicar nos botões mesmo em movimento.

Navegação Direta: A tela inicial (Home) funciona como um hub central, permitindo acessar o mapa, o cardápio ou os replays com apenas um toque.

Contraste: A organização dos elementos prioriza a leitura rápida, essencial para quem está dividindo a atenção entre o celular e o campo.

Fluxo do Usuário (Baseado no Protótipo)
Localização: Na tela inicial, o usuário acessa o Mapa para visualizar sua posição real no setor, facilitando a busca por assentos e banheiros.

Pedido de Comida: O usuário navega pelo Cardápio, adiciona os itens (como Hot Dog ou Água) ao Carrinho e visualiza o resumo antes de confirmar a retirada.

Ver Replay: Através da Central de Replays, é possível selecionar um lance específico e escolher entre diferentes ângulos de câmera (Câmera 1, 2 ou 3).
